# 👟 Step Up – Online Shoe Store

A modern, fast, and fully responsive e-commerce website for buying shoes online. Built with a clean UI, smooth navigation, and an easy shopping experience across mobile, tablet, and desktop devices.

## 🚀 Features

- 🏠 **Home Page** – Hero banner with offers, featured products, and categories
- 🛒 **Product Listing** – Filter by brand, price, size, color, and category (Men/Women/Kids)
- 👟 **Single Product Page** – Multiple images, size/color selector, related products
- 🛍️ **Cart Page** – Update quantity, remove items, view total price
- 💳 **Checkout Page** – Shipping form with order summary
- ❤️ **Wishlist** – Save favorite products for later
- 🔍 **Live Search** – Instantly filter products as you type
- 📱 **Fully Responsive** – Optimized for mobile, tablet, and desktop
- ⚡ **Fast Loading** – Optimized images and minimal heavy assets

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript
- (Optional) React.js for dynamic components
- LocalStorage / React State for cart management

## 🔧 Installation & Usage

1. Clone the repository
```bash
   git clone https://github.com/your-username/shoe-store.git
```
2. Navigate to the project folder
```bash
   cd shoe-store
```
3. Open `index.html` in your browser, or run with a live server extension (e.g. VS Code Live Server)
